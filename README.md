# ToDoPlan (HarmonyOS ArkTS)

一个使用 **HarmonyOS ArkTS** 编写的待办/专注类应用示例项目。

## 主要功能（部分仍在迭代）

- 计划创建与管理
- AI 对话（流式响应解析）
- 未来倒计时
- 自习室 / 许愿墙（SQLite 持久化、许愿/还愿、编辑、右滑删除）

## 运行与构建

使用 DevEco Studio / hvigor 构建运行本项目。

- **打开工程**：选择项目根目录
- **构建**：执行 hvigor 构建任务（由 IDE 驱动）
- **运行**：连接设备或使用模拟器运行

> 具体任务名称会随你的 DevEco Studio / SDK 版本略有差异，以 IDE 的 Build/Run 为准。
